Requerimientos

Para poder ejecutar este codigo, el cual esta implementado en JAVA, es necesario tener un IDE de desarrollo que soporte el lenguaje de programacion JAVA, algunos de los que se puede utilizar son:

  • Netbeans: https://netbeans.org/downloads/

  • Eclipse: https://www.eclipse.org/downloads/

Una vez en ellos, debemos crear un proyecto con el nombre que les plazca. Declaramos un paquete llamado filosofos y creamos dos clases, cada una con el nombre que se indica en las pestañas donde se encuentra el código.

  • El codigo se ejecuta hasta el fin, asi que es necesario parar la ejecucion antes de cerrar la consola.
  • El codigo se encuentra debidamente comentado de tal forma que es de fácil entendimiento de la funcionalidad de cada línea de código.
